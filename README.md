# 实验室样本流转登记工具

本地桌面版实验室样本全生命周期流转管理系统，支持从样本接收到分装、复核、退回的完整流程追溯。

## 功能特性

### 核心流程
- **样本接收**：创建批次、录入唯一编号、登记数量和来源
- **分装管理**：对已接收样本进行分装操作
- **提交复核**：分装完成后提交给复核员审核
- **复核通过（交接登记）**：复核员审核通过，自动记录交接人（复核员）和交接时间，样本进入可交接状态
- **退回处理**：任意环节发现问题可退回，并记录退回原因
- **撤销退回**：退回后可撤销最近一次状态变更，**保留完整退回历史记录**

### 业务规则
- **编号唯一性**：同一批次内样本编号不可重复，重复保存会被拒绝
- **权限控制**：普通操作员不能直接执行复核通过操作
- **状态追溯**：每次操作记录操作人、时间、原因和备注
- **数据持久化**：所有数据本地存储，程序重启后数据完整保留

## 技术栈

- **Electron** - 桌面应用框架
- **React 18** - UI 框架
- **TypeScript** - 类型安全
- **Vite** - 构建工具
- **本地 JSON 文件** - 数据持久化存储

| 复核员老李 | 复核员 | 拥有全部权限，含复核通过 |

## 验证步骤

### 场景一：完整正向流程（收样 → 分装 → 复核 → 导出）

1. **创建批次**
   - 打开「样本接收」页面
   - 点击右上角「+ 新建批次」
   - 输入批次编号：`BATCH-20240616-001`
   - 输入批次名称：`6月16日第一批送检`
   - 点击「创建」

2. **接收样本**
   - 在批次下拉框中选择刚创建的批次
   - 输入样本编号：`SAMPLE-001`
   - 输入数量：`5`
   - 输入来源：`内科病房`
   - 点击「确认接收」
   - 下方列表显示已接收的样本

3. **分装操作**
   - 切换到「样本列表」页面
   - 找到刚才的样本，状态为「已接收」
   - 点击「分装」按钮
   - 填写备注：`分装为5管`
   - 点击「确认」
   - 样本状态变更为「已分装」

4. **提交复核**
   - 点击「提交复核」按钮
   - 确认操作
   - 样本状态变更为「待复核」

5. **复核通过**
   - 在左下角用户切换处，切换到「复核员老李」
   - 找到待复核的样本
   - 点击「复核通过」按钮
   - 确认操作
   - 样本状态变更为「已复核通过」

6. **导出交接清单**
   - 切换到「交接导出」页面
   - 选择对应批次
   - 点击「导出 CSV」
   - 选择保存位置，保存文件
   - 用 Excel 打开导出文件，验证内容正确

### 场景二：重复编号校验（失败链路）

1. 选择一个已有样本的批次
2. 尝试录入与已有样本相同的编号
3. 系统提示「同一批次中已存在相同的样本编号，无法保存」
4. 验证：修改编号为新值后可正常保存

### 场景三：权限控制（失败链路）

1. 确保当前用户为「操作员小王」（普通操作员）
2. 将一个样本推进到「待复核」状态
3. 尝试点击「复核通过」
4. 验证：按钮不会出现（普通操作员看不到复核通过按钮）
5. 切换到「复核员老李」
6. 验证：可以看到「复核通过」按钮并执行操作

### 场景四：退回与撤销（失败链路）

1. 选择一个「已分装」状态的样本
2. 点击「退回」按钮
3. 填写退回原因：`样本破损`
4. 填写备注：`第3管有裂痕`
5. 点击「确认」
6. 样本状态变更为「已退回」
7. 点击「历史」按钮，查看完整流转记录
8. 点击「撤销退回」按钮
9. 样本状态恢复为「已分装」
10. 再次查看历史，验证撤销后记录正确

### 场景五：数据持久化验证

1. 完成上述操作后，关闭程序
2. 重新启动程序
3. 验证：
   - 批次信息完整保留
   - 所有样本状态与关闭前一致
   - 历史记录完整可查
   - 交接人、数量等信息准确

### 场景六：历史追溯验证

1. 对任意样本执行多次状态变更
2. 点击「历史」按钮
3. 验证每条记录包含：
   - 操作时间
   - 操作人
   - 操作动作
   - 状态变化（从 xx 到 xx）
   - 原因和备注（如有）

## 数据存储位置

程序数据存储在系统用户数据目录下：

- **Windows**: `%APPDATA%/lab-sample-tracker/data/lab-sample-data.json`
- **macOS**: `~/Library/Application Support/lab-sample-tracker/data/lab-sample-data.json`
- **Linux**: `~/.config/lab-sample-tracker/data/lab-sample-data.json`

## 状态流转图

```
已接收 → 已分装 → 待复核 → 已复核通过（可导出）
  ↓        ↓        ↓
  ←退回←  ←退回←  ←退回←
           ↓
        已退回 → 可撤销回上一状态
```
